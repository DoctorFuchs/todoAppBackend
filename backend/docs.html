API documentation

  1. Creating a user

    Make Http/Https(depending on server config) request to [your domain/ip]/createUser
    Required body data in JSON format:

    1. username in field "u"
    2. email in field "m"
    3. HASHED password in field "pwh" (sha512 unsalted)

  2. Removing a user

    Make Http/Https(depending on server config) request to [your domain/ip]/removeUser
    Required body data in JSON format:

    1. email in field "m"
    2. Hashed password in field "pwh"(has to be the same used to create user)
